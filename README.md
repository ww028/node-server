# node.js 和 mysql 开发接口
> /controller 接口文件
>   /name.js 接口
> /mysql 数据库
>   /config copy.js 数据库配置
>   /config.js 数据库配置，这个文件没有上传，修改config copy.js 文件名为config.js
>   /pool.js 数据库连接池
> /router
>   index.js 路由，配置接口地址
> 
> /mysql/config.js 这个文件有私密信息所以没上传
> /mysql/config copy.js 把文件名的 copy 删掉，然后在文件内填上响应的配置就可以了
