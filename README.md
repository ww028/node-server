# node.js 和 mysql 开发接口
``` bash
# install dependencies
npm install

# start
npm start
```
> /controller 接口文件   
>>   /name.js 接口   
> /mysql 数据库   
>>   /config copy.js 数据库配置 </br>
>>   /config.js 数据库配置，这个文件没有上传，修改config copy.js 文件名为config.js </br>
>>   /pool.js 数据库连接池 </br>
> /router </br>
>>   index.js 路由，配置接口地址 </br>
